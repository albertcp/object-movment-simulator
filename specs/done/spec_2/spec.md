# Requirement 2

Possibility to modify speed object when created via a speed edition button.

How it works:
1. When an object is created on the map, the user can click an object and then click the edit speed button and a popup with the default speed will appear.
2. The user can erase the speed and insert the new one. The speed will be show on meters per second.
3. The user will set the new speed and when the user press start, then the object will move at that speed
