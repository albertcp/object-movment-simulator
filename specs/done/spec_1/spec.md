# Requirement 1

Button `Add object` can be pressed more than 1 time to add more than 1 object. 

How it works:
1. Press `Add object` Button
2. Then, operator needs to select a path for the object to follow.
3. When push button is pressed again. The user has defined the path for
the one object.
4. The operator can pressed again the same button to add a new object.
5. Note: meanwhile the operator is drawing the object, the add object pushbutton
is locked unitl pressed again.
